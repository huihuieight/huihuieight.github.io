Chrome安装React developer tools

1、安装Git；

2、在一个空文件夹中，右键选择Git Bash Here，进入Git界面，输入
git clone https://github.com/facebook/react-devtools.git，该指令是从github下载react-devtools到本地，下载完成后能看到空文件夹中多了一个react-devtools文件夹；

3、在刚刚的git窗口中输入cd react-devtools，（或者关闭原先的git窗口，直接双击进入react-devtools文件夹，右键选择Git Bash Here），这一步是保证在react-devtools文件夹执行git指令，同时会发现该文件夹下的文件很少（原因在第4点）。
使用淘宝镜像，会在安装依赖时快速一点。
npm --registry https://registry.npm.taobao.org install

4、通过github的react-devtools源码readme.md可以看到The source code for the v3 of the extension can be found in the v3 branch.源文件在v3分支上，输入指令git checkout v3，可以看到本地文件夹中的文件变的很多，说明获取到了正确的源码；

5、输入npm install，安装依赖，等待一会；
6、打包扩展程序：npm run build:extension:chrome

7、打包成功后，进入chrome浏览器，右上角“更多工具”–>扩展程序–>加载已解压的扩展程序，选择react-devtools -> shells -> chrome -> build -> unpacked文件夹，加载完成后浏览器右上角会出现react的图标，重启浏览器就能进行react的调试工具了。
